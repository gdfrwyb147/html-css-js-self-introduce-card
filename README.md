# HTML/CSS 个人简介卡片 🎉
这是我自学前端的第一个实战小项目，从0到1完成了一个完整的个人介绍网页。
链接
##https://gdfrwyb147.github.io/html-css-js-self-introduce-card/index.html

## 项目介绍
- **技术栈**：纯 HTML + CSS
- **功能**：展示个人头像、基本信息、技能和自我介绍
- **核心实现**：
  - 圆形头像样式（border-radius）
  - 页面水平居中（margin: 0 auto）
  - 舒适的文字排版（行高、间距调整）
  - 解决了图片路径、元素居中等新手常见问题

## 学习收获
1.  掌握了 HTML 基础标签的使用（`<img>`, `<p>`, `<strong>` 等）
2.  理解了 CSS 核心布局与美化属性（margin, padding, font-size, line-height）
3.  第一次尝试用 GitHub 管理和保存自己的学习成果
4.  学会了调试页面显示问题（图片不显示、排版错乱等）
